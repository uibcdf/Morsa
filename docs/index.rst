=====
Morsa
=====

Morsa es una sencilla librería que permite simular competiciones entre jugadores de tres tipos
-ballenas, morsas y pingüinos- que se enfrentan entre si en partidas de ajedrez, dominó, tres en
raya y parchís. ¿Será que alguno de los tres animales tiene mas posibilidades de ganar que los
otros?

Esta librería está creada únicamente con el propósito de ilustrar la creación de una librería de Python para uso público y de código libre, con documentación y estrategia de testeo.
El tutorial que describe el desarrollo de Morsa se encuentra en `la sección "Flujo de Trabajo" del repositorio Academia de la UIBCDF <https://github.com/uibcdf/Academia#el-flujo-de-trabajo-en-la-uibcdf->`_ como la unidad titulada `"Creando Librería de Python" <https://github.com/uibcdf/Academia/blob/tutorial_sphinx/Flujo_de_trabajo/Creando_Libreria_Python/creando_libreria_python.md>`_.
